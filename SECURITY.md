# Security Policy

## Supported Versions

We currently support the following versions of HelixStudio AI:

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to **security@helixstudio.ai** (randy@lucycoreai.com or scottymicfree@gmail.com). All security vulnerabilities will be promptly addressed.

Please include:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.

We ask that you do not disclose the vulnerability publicly until we have had a chance to fix it.
